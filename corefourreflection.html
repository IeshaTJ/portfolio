Loop

A loop is a sequence of instruction s that is continually repeated until a certain condition is reached. Typically, a certain process is done, such as getting an item of data and changing it, and then some condition is checked such as whether a counter has reached a prescribed number. Python Syntax: edibles = ["taco", "spam","fries","burger"] for food in edibles: if food == "spam": print("No more spam please!") break print("Great, delicious " + food) else: print("I am so glad: No spam!") print("Finally, I finished")

variables

A variable in coding is a storage location paired with an associated symbolic name (an identifier), which contains some known or unknown quantity of information referred to as a value.
